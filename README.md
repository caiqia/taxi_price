# taxi_price
Ce projet est une application prototype qui calcule le prix d'un trajet pour l'ensemble de leur flotte de taxis.

### Exécution
back-end: Allez à la répertoire backend
```
cd backend
npm start
```
front-end: Allez à la répertoire my-app
```
cd my-app
npm start
```
### Test
```
cd my-app
npm test
```
